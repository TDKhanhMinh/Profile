# Portfolio - Trần Đỗ Khánh Minh

Personal portfolio built with Next.js, TypeScript, Tailwind CSS, Framer Motion, and Lucide React.

## Getting Started

```bash
npm install
npm run dev
```

Open `http://localhost:3000` to view the site.

## Tech Stack

- Next.js 16 App Router
- React 19
- TypeScript
- Tailwind CSS v4
- Framer Motion
- Lucide React

## Deploy

Deploy on Vercel from the main GitHub branch.
